# 🚀 MISSION: BIG BOT
### AI-Powered Equity Auditor for the Indian Market

**BIG BOT** is a specialized dashboard built to provide professional-grade financial insights using the **Gemini 3.1 Flash** model. It combines real-time data from the NSE with advanced AI reasoning to help investors make data-driven decisions.

---

## 🛠️ Tech Stack
* **Engine:** Python 3.12
* **Brain:** Google Gemini 3.1 Flash
* **Interface:** Streamlit Cloud
* **Data Source:** Yahoo Finance (NSE/BSE)

## 🔐 Security Note
This app uses **Streamlit Secrets** and encrypted sidebar inputs to manage API keys. Your data and AI credits remain private and secure.
